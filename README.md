# task-tracker
Task tracker web app using Servlet, JSP and Java JDBC

**Prerequisites**
1. JDK 21
2. Apache Tomcat 10.1.12
3. PostgreSQL

**Installation**
1. Clone the repository
2. Configure the database in database.properties file
3. Run the project using Intellij IDEA
